## Root-Level References

The repository root is reserved for project entry documents and a small number
of operational files. Historical reports should live here in `docs/` or a
subfolder under it rather than beside the main readme.

## Maintenance Rule

When adding a major report, exported artifact, or planning document, place it
in `docs/` and update this index if the file is meant to be discoverable.
