# NEXUS-R Roadmap

## ✅ v0.1 — Foundation (Current)
*June 2026*

**Goal:** Working local-first agent runtime with core routing, UI, and deployment.

- [x] Multi-provider routing (10+ providers)
- [x] Local-first architecture with privacy-by-default
- [x] Web dashboard (React 19 + TypeScript)
- [x] Persistent memory (SQLite + ChromaDB)
- [x] Real-time model streaming
- [x] Permission tiers (T1–T5)
- [x] Audit trail (append-only event log)
- [x] Docker Compose deployment
- [x] CI/CD pipeline (lint, typecheck, test, security, Docker)
- [x] Full documentation (architecture, contributing, FAQ, troubleshooting)

---

## 🎯 v0.2 — Enterprise Features (Q3 2026)

**Goal:** Authentication, compliance exports, and production hardening.

- [ ] SSO / SAML / OIDC authentication
- [ ] Role-based access control (RBAC)
- [ ] Structured audit export (CSV, PDF, JSON)
- [ ] Usage analytics dashboard
- [ ] Prometheus metrics + Grafana dashboards
- [ ] Health checks + auto-recovery
- [ ] Plugin system for custom providers

---

## 🎯 v0.3 — Collaboration (Q4 2026)

**Goal:** Team workspaces, sharing, and managed deployment.

- [ ] Team workspaces with shared conversations
- [ ] Conversation branching and comparison
- [ ] One-click deploy to AWS/GCP/Azure
- [ ] API key management UI
- [ ] Workflow templates (50+ pre-built)

---

## 🎯 v1.0 — Platform (H1 2027)

**Goal:** Mature platform with ecosystem and enterprise readiness.

- [ ] Plugin marketplace for community providers
- [ ] Workflow templates (100+)
- [ ] SOC 2 Type II certification
- [ ] Multi-region deployment
- [ ] Tool/function calling support
- [ ] RAG pipeline with document upload
