Workflow Engine module contract:
1. Record causal execution traces.
2. Preserve step order.
3. Link steps by parent event id.
4. Capture tool, action, model, cost, and verification.
5. Support trace retrieval by task id.
6. Provide lightweight summary views.
7. Avoid ETD extraction in Phase 1.
8. Avoid hidden side effects.
9. Rely on State Core persistence.
10. Preserve data needed for Phase 2 evolution.
