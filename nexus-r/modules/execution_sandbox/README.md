Execution Sandbox module contract:
1. Run only workspace-scoped actions.
2. Support filesystem list, read, write, append, and search.
3. Support restricted terminal commands.
4. Enforce path confinement.
5. Deny destructive operations.
6. Log every invocation and outcome.
7. Prefer subprocess fallback on Windows.
8. Optionally support Docker later.
9. Expose deterministic execution results.
10. Reserve browser execution for Phase 3.
